# 🎮 Jogo da Forca

Projeto simples em Python que simula o clássico jogo da forca.

## 📌 Funcionalidades
- Palavras aleatórias
- Controle de tentativas
- Verificação de letras corretas e incorretas
- Condição de vitória e derrota

## 🛠️ Tecnologias
- Python
- Biblioteca random

## 🎯 Objetivo
Praticar lógica de programação, manipulação de strings e estruturas de repetição.

## ▶️ Como executar
Execute o arquivo `main.py` em um interpretador Python.
